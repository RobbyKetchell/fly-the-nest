## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository focused on Docker and Auth.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# build docker
$ npm run docker:build

# run docker
$npm run docker:run
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
